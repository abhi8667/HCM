\begin{abstract}
This report summarizes the complete lifecycle of the HCM pathogenicity classification project, from data preparation and leakage prevention to model development, cross-gene validation, interpretability, external benchmarking, and VUS re-stratification. The implemented pipeline combines tabular biophysical features with ESM-2 language-model embeddings in a Two-Tower architecture and compares performance against a Random Forest baseline and external EVE scores. The project delivers reproducible scripts, trained model artifacts, benchmark outputs, and manuscript-ready figures/tables.
\end{abstract}

\section{Project Overview}
\subsection{Clinical Motivation}
Hypertrophic Cardiomyopathy (HCM) is strongly associated with missense variants in sarcomeric genes. A major bottleneck in clinical interpretation is the large number of variants classified as VUS (Variant of Uncertain Significance). This project addresses that gap by building an unbiased machine learning pipeline for pathogenicity classification and downstream VUS prioritization.

\subsection{Primary Objective}
Build and validate a robust, low-leakage predictive framework that:
\begin{itemize}[leftmargin=1.2cm]
    \item Classifies variants as pathogenic or benign,
    \item Generalizes across genes using Leave-One-Gene-Out (LOGO) validation,
    \item Produces calibrated probabilities for practical ranking,
    \item Supports mechanistic interpretation through in silico mutagenesis,
    \item Re-stratifies VUS variants in a larger unlabeled set.
\end{itemize}

\section{Methodology and Work Completed}
\subsection{Month 1: Data Cleaning and Embedding Preparation}
\textbf{Completed work:}
\begin{itemize}[leftmargin=1.2cm]
    \item Implemented leakage-aware cleaning by dropping potentially leaky clinical/meta fields (e.g., population and review-derived columns).
    \item Constructed mutation-aware sequence handling for wild-type and mutant context windows.
    \item Extracted delta ESM-2 embeddings and persisted them for downstream reuse.
    \item Established baseline LOGO workflow using a Random Forest classifier.
\end{itemize}

\textbf{Deliverables:}
\begin{itemize}[leftmargin=1.2cm]
    \item \texttt{data/esm2\_delta\_embeddings.npy}
    \item Baseline artifact \texttt{models/hcm\_logo\_baseline\_model.joblib} (as referenced in reproducibility docs)
\end{itemize}

\subsection{Month 2: Two-Tower Modeling, Evaluation, and Interpretation}
\textbf{Completed work:}
\begin{itemize}[leftmargin=1.2cm]
    \item Implemented a Two-Tower neural architecture (tabular tower + ESM tower + fusion head).
    \item Trained under class imbalance using weighted BCE.
    \item Executed LOGO validation across all 9 genes.
    \item Applied probabilistic calibration and reported Brier score and ECE.
    \item Trained and evaluated a Baseline Random Forest comparator per split.
    \item Ran ablation checks and generated in silico mutagenesis landscapes.
\end{itemize}

\textbf{Deliverables:}
\begin{itemize}[leftmargin=1.2cm]
    \item \texttt{results/logo\_metrics.csv}
